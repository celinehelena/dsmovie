# DSMovie


Projeto de aplicação full-stack de avaliação de filmes e séries. Desenvolvido durante a Semana Spring React do devSuperior.


## Tecnologias 
Tecnologias e ferramentas utilizadas para o desenvolvimento do projeto:

- React
- Next.js
- TypeScript
- React Icons
- Tailwind CSS
- React Hook Form
- Java
- Spring Boot
- H2
- PostgreSQL
